# 🎮 Menu de Jogos em Portugol | Game Menu in Portugol

## 🇧🇷 Português

### 📖 Sobre o projeto

Este projeto foi desenvolvido em **Portugol** como parte das atividades da disciplina de programação, em conjunto com a turma e o professor.

O objetivo foi criar um **menu interativo de jogos**, permitindo que o usuário escolha qual jogo deseja executar por meio de uma interface simples no terminal.

### 🎯 Jogos disponíveis

* 🐍 Jogo da Cobrinha
* 🚀 Jogo da Nave
* ❌⭕ Jogo da Velha

Cada opção do menu direciona o usuário para o respectivo jogo.

### 🛠️ Tecnologias utilizadas

* Portugol

### 📚 Objetivos de aprendizagem

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Estruturas condicionais
* Estruturas de repetição
* Procedimentos/Funções
* Organização do código
* Modularização
* Lógica de programação
* Trabalho em equipe

### ▶️ Como executar

1. Abra o projeto no seu interpretador de Portugol (como o Portugol Studio).
2. Execute o arquivo principal.
3. Escolha um dos jogos disponíveis no menu.
4. Divirta-se!

### 👨‍🏫 Projeto acadêmico

Projeto desenvolvido para fins educacionais, com orientação do professor e participação dos alunos da turma.

This project was developed in **Portugol** as part of a programming course, together with our class and professor.

The main goal was to create an **interactive game menu**, allowing users to choose which game they want to play through a simple terminal interface.

## 🎯 Available games

* 🐍 Snake Game
* 🚀 Space Shooter
* ❌⭕ Tic-Tac-Toe

Each menu option launches its corresponding game.

* Conditional statements
* Loops

* Code organization

## ▶️ How to run

1. Open the project in a Portugol interpreter (such as Portugol Studio).
